# CRUD-N-M

## Project setup
```
yarn install
```

### Hot-reloads for development
```
yarn run dev
```

### Production
```
yarn run start
```

### Insomnia configuration file
```
Application>Preferences>Data>ImportData>FromFile>CRUD_Insomnia.json
```
### Warning
```
Create a database named, CRUD_N_M (in Docker or on the local machine)
```
